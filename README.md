## Backend I – Entrega 1

API desarrollada con Node.js y Express para la gestión de productos y carritos de compra.

## Funcionalidades

- Productos
- Listar productos
- Obtener producto por ID
- Crear producto
- Actualizar producto
- Eliminar producto
- Carritos
- Crear carrito
- Obtener productos de un carrito
- Agregar productos a un carrito

## Utilizacion de metodos

GET / POST / PUT / DELETE de products y carts

Tecnologías utilizadas

- Node.js
- Express
- File System (fs)
